Checkbox

Checkbox elements, by default, are wrapped with a label element and a .checkbox class. Be sure to add type="checkbox" as an attribute.

A basic checkbox has an input wrapped by a label.

It is best practice to enclose checkbox inputs inside of an form-group element. Wrapping the checkboxes keeps the corresponding question inline with the checkbox elements.

Demo:
Markup:
<div class="form-group">
  <label class="checkbox" aria-label="">Question:</label>
  <label class="checkbox" for="">
    <input type="checkbox" id="" name="">Checkbox 1
  </label>
  <label class="checkbox" for="">
    <input type="checkbox" id="" name="">Checkbox 1
  </label>
</div>
