Radio

Radio elements, by default, are wrapped with a label element and a .radio class. Be sure to add type="radio" as an attribute.

A basic radio has an input wrapped by a label.

It is best practice to enclose radio inputs inside of a form-group element. Wrapping the radios keeps the corresponding question inline with the radio elements.

Demo:
Markup:
<div class="form-group">
  <label class="radio" aria-label=""></label>
  <label class="radio" for=""Question:>
    <input type="radio" name="" id="" checked>Radio 1
  </label>
  <label class="radio" for="">
    <input type="radio" name="" id="" checked>Radio 2
  </label>
</div>

Radio Group Stacked

If you'd like to use this in a form or with a label for a question it is wrap each section in a form-group and put the question label inside. The radio inputs are already wrapped by their label.

Markup:
<div class="form-group--stacked">
  <label class="radio" for="">
    <input type="radio" name="" id="" checked>Radio 1
  </label>
  <label class="radio" for="">
    <input type="radio" name="" id="" checked>Radio 2
  </label>
</div>
