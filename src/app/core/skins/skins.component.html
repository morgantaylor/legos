Skin

Skins are global classes that can be used on components. Visit the individual component pages to see how to implement them.

Colors

  • skin-{{ color }}

Skins Control:

  • Background Color
  • Font Color
  • Border Color (optional add-on)

Components:

  • Select Input
  • Accordion
  • Breadcrumb
  • Switch

Not Available For:

Some components don't use global skins. Those components need more specific styles to facilitate their skins. The below components have special skin classes. See their pages for more information.

  • Button
  • Chips
  • Tab
  • Pagination
