Add to Your Project

Adding LegoCSS to your project is as easy as downloading and including the minified file in your project or using our hosted version.

Add the following <head> inside the <head> tag before any other stylesheets.

<link rel="stylesheet" href="https://legocss.com/build/lego.min.css"
