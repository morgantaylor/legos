Dropdown

Our dropdowns are built with CSS without any need for Javascript. Under the hood, they are animated with a hidden input checkbox. Because of this, if the dropdown is open, click it again to close it. Clicking outside of the dropdown will not close it, but adding some custom javascript could easily change that.

Markup:
<div class="dropdown">
  <input type="checkbox" class="dropdown__toggle" aria-label="dropdown__toggle">
  <button type="button" class="dropdown__btn btn btn--outline-blue">
    <span>Dropdown Menu</span>
    <i class="icon fas fa-caret-down" aria-hidden="true"></i>
  </button>
  <ul  class="dropdown__container">
    <li class="dropdown__item">...</li>
  </ul>
</div>

To disable a dropdown component two things need to be happen. The input element needs to be :disabled as well as a .disabled class added to the dropdown component.

<div class="dropdown disabled">
  <input type="checkbox" disabled class="dropdown__toggle" aria-label="dropdown__toggle">
  ...
</div>
