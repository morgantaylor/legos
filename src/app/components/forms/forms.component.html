Form

Forms are a large part of our daily lives. We spend our time filling out a contact form, adding items to a shopping cart, and taking a survey. Those form elements and how they all fit together are critical to web design and user experience.

Attributes:
  • for : label element
  • id : input element
  • type : input element
  • placeholder : input element
  • aria-label, title : input element, defines a label for the element (only used when label text not visible on screen)
  • type : button element

Accessibility

Forms are made up of input, label, and button elements. Match label elements with its associated input element using the for and id attributes. Utilize the placeholder attribute for certain input elements that need to be given a hint.

Form Group

The .form-group class combines a label and input element. This is not needed for small inline forms. It is intended for lengthier --stacked or --aligned forms.

Labels

Don't use a div or p element instead of a label element in a form. The .form__label properties will override a paragraph element's characteristics but it is best practice to write semantic code.

Demo:
Markup:
<form class="form--inline">
  <label for="name">Name:</label>
  <input id="name" type="text" placeholder="...">
  <label for="email">Email:</label>
  <input id="email" type="email" placeholder="...">
  <button class="btn btn--blue" type="button">Submit</button>
</form>

Stacked

To create a stacked form with input elements below their corresponding label elements, add --stacked to the .form class on the form element.

Demo:
Markup:
<form class="form form--stacked">
  <div class="form-group">
    <label for="name">Name:</label>
    <input id="name" type="text" placeholder="...">
  </div>
  ...
</form>

Aligned

Three alignment classes are available, -left, -middle, and -right. To add an alignment to a form, add .form--aligned-[direction] to the form element. On smaller screens, the aligned forms will switch to a --stacked form for a better user experience.

Demo: Aligned-center
Demo: Aligned-right
Markup:
<form class="form form--aligned-[center]">
  <div class="form form-group">
    <label for="name">Name:</label>
    <input id="name" type="text" placeholder="...">
  </div>
  ...
</form>

Legends

Visually group related elements within the fieldset element. An optional legend can be used to give the group a name and border.

Demo:
User
Markup:
<form class="form form--stacked">
  <fieldset>
    <legend>User</legend>
      <div class="form-group">
        <label>Username</label>
        <input type="text" placeholder="...">
      </div>
      <div class="form-group">
        <label>Password</label>
        <input type="password" placeholder="...">
      </div>
      <button class="btn btn--green" type="submit">Submit</button>
  </fieldset>
</form>
