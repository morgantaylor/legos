Button Group

Buttons grouped as a toolbar are called button groups. To create a button group wrap a collection of buttons in a .btn-group. Add skins to the buttons just as you normally would.

Demo:
Markup:
<div class="btn-group" role="group" aria-label="...">
  <button type="button" class="btn btn-group__btn btn--outline-default">Middle</button>
  <button type="button" class="btn btn-group__btn btn--outline-default">Left</button>
  <button type="button" class="btn btn-group__btn btn--outline-default">Right</button>
</div>
