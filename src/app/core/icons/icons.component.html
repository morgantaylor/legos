Icon

LegoCSS uses Font Awesome. Visit the website to see all the available icons. You can use these icons anywhere using a span or i element. To learn more about accessibility read their article.

Attributes:
  • aria-hidden : 'true'
  • title : adding a title provides a tooltip for sighted mouse users
Demo:
Markup:
<i class="icon fas fa-shower"></i>

Popular:

fa-{{ popularIcon }}

Decorative:

Decorative icons have no functional purpose and will not affect user interacts. Add an aria-hidden attribute to the icon, alerting screen readers to not read the element.

Demo:
Markup:
<i class="icon fas fa-shower" aria-hidden="true"></i>

Semantic:

A semantic icon is used to convey meaning, such as inside a button with no associated text. To make this icon accessible for assistive technologies there are a few things you need to do.

If the icon is wrapped by another element such as a button or anchor element add an aria-label to the outer element. If the icon is by itself and not wrapped by another element add the aria-label to the icon element itself.

Attributes:
  • aria-label : provide an accessible alternative name for the element
Demo:
Markup:
<button ... aria-label="" title="">
  <i ... aria-hidden="true"></i>
</button>

OR

<i ... aria-hidden="true" aria-label="" title=""></i>
