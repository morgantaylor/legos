Breadcrumb

Navigation is at the core of user experience. Breadcrumbs enable a user to understand where they've been, how they got there, and show a way back if necessary. It can be infuriating if you’ve ever wanted to go back one or two screens but haven’t been given an option to do so. You find yourself clicking the home button and end up losing your place. Breadcrumbs prevent those situations from happening.

Best practice: Use an ordered list for this component. It is a list and it does matter what order the list is in.

Demo:
  1. Home
  2. About Us
  3. Section 1
Markup:
<ol class="breadcrumb">
  <li class="breadcrumb__item">
    <a href="#">Home</a>
  </li>
  <li class="breadcrumb__item">
    <a href="#">Library</a>
  </li>
  <li class="breadcrumb__item active">
    Data
  </li>
</ol>
