Text Formatting

This is not an exhaustive list of all text formatting elements but some of the most commonly used ones.

<strong> Contents have strong importance
<em> Text that has stress emphasis
<s> text with a strikethrough, or a line through it
<small> small print, like copyright and legal text
<sup> Normal TextSuperscript text
<sub> Normal TextSubscript text
<code> Short fragment of computer code
<kbd> Cmd + C Textual user input from a keyboard
<samp> Represents sample output from a computer program
<mark> Text which is marked or highlighted
