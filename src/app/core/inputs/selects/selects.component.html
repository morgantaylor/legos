Select

Native select menus, namely in Safari and Chrome, are not customizable. They have rounded corners that cannot be modified with CSS which can be frustrating. This select component customizes the border-radius, color, and arrow icons.

To disable a select component the inner select element needs to be :disabled, and a .disabled class added to the select component wrapper.

Attributes:
  • name : select element, specifies the name of the control group
  • multiple : select element, indicates whether more than one option is allowed to be selected
  • label : option element, if there is no label present then the value attribute becomes the label text
  • value : option element, the value to be submitted with the form
  • aria-label : defines a label for the element (only used when label text not visible on screen)

By default, the select component is a inline element, just like input and button elements and takes up only the space it needs.

Demo:
Markup:
<div class="select">
  <select name="name" id="id">
    <option>Choose an option</option>
    <option value="1"></option>
    <option value="2"></option>
    <option value="3"></option>
  </select>
</div>

Stacked

To make a select component stack ontop of another component add --stacked to the .select class wrapper.

Demo:
Markup:
<div class="select--stacked">
  <select name="name" id="id">
    ...
  </select>
</div>

Squared Corners

To change the border-radius on the select component to be squared add .select--squared to the select wrapper.

Demo:
Markup:
<div class="select select--squared">
  <select name="name" id="id">
    ...
  </select>
</div>
