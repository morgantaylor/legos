Pagination

The pagination component is for showing multiple pages of related content.

Attributes:
  • role : 'navigation'*
  • aria-label : defines a label for the element (only used when label text not visible on screen)

*Note: The role attribute needs to be used for the nav element as well as the items inside. There are many roles to choose from, see the full list.

Accessibility

For accessibility purposes, the pagination component is a nav element. It is also possible and highly likely that a page may have more than one navigation, such as the primary navigation in the header or sidebar. It is best practice to have a descriptive aria-label on the nav element. This will signal to screen readers and other assistive technologies what the element is.

Example: If the pagination component is used to navigate between a set of pages in a signup form, the label could be aria-label="sign up form pages".

Markup:
<nav class="pagination" role="navigation" aria-label="Pagination Navigation">
  <span class="pagination__item pagination__item--first [ifDisabled]" role="button">
    <a href="" aria-label="[Previous or < symbol]">First</a>
  </span>
  <span class="pagination__item" role="button">
    <a href="">&#60;</a>
  </span>
  <span class="pagination__item" role="button" class="active">
    <a href="">1</a>
  </span>
  <span class="pagination__item" role="button">
    <a href="">2</a>
  </span>
  <span class="pagination__item" role="button">
    <a href="">3</a>
  </span>
  <span class="pagination__item" role="button">
    <a href="">&#62;</a>
  </span>
  <span class"pagination__item pagination__item--last" role="button">
    <a href="" aria-label="[Next or > symbol]">Last</a>
  </span>
</nav>

Prev/Next Labels

Customize how to show the previous/first and next/last items. Either choose to have all of them displayed, or just choose one set. Either way, both symbols, and words are allowed. Adding --first or --last to a .pagination__item gives them a margin that separates them from the inner pagination buttons.

Demo:
First « 1 2 » Last
Markup:
<nav class="pagination" role="navigation">
  <span class="pagination__item pagination__item--first" role="button">
    <a href="" aria-label="First">First</a>
  </span>
  <span class="pagination__item" role="button">
    <a href="" aria-label="[Previous or < symbol]">«</a>
  </span>
  ...
  <span class="pagination__item" role="button">
    <a href="" aria-label="[Next or > symbol]">»</a>
  </span>
  <span class="pagination__item pagination__item--last" role="button">
    <a href="" aria-label="Last">Last</a>
  </span>
</nav>

Skins

All basic skins are available for this component.

Add the skin class to the main pagination element for the class to take effect.

*Note: The gray skin is not available for the pagination component. The default color is already gray.

Demo: pagination--{{ color }}

Markup:
<nav class="pagination pagination--{{ '{' }}{{ '{' }} color {{ '}' }}{{ '}' }}" role="navigation">
