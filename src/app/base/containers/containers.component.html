Container

Quickly added the layout of content with these three classes. The intention of these classes is to be used for only simple base layouts. For more complex containers you may need to add more explicit CSS.

Base Container

The .container class creates a responsive box with space on the bottom. The max-width of these containers is 700px. This width improves paragraph readability while also providing padding for mobile devices.

*Note: If you nested .container classes 25px of padding will be added to the left side of the inner container. This is intended to be used for indenting content subsections or dividing parts of a form.

Markup:
position: relative
  margin: 0 auto 20px auto
  padding: 0
  width: 100%
  max-width: 700px
  padding: 0 20px

Flex Row

Add the .flex-container with the .container class to make any .container a flex parent (default row). This class can be used with or without the .container class. This flex container uses the below properties.

Markup:
display: flex
align-items: center
flex-wrap: wrap

Flex Column

Add the .flex-column class with .container class to transform the container into a flex parent with a column direction. This class does not share styles with the .flex-container class. Similar to the .flex-container class this class can be used with or without the .container class. This flex container uses the below properties.

Markup:
display: flex
flex-direction: column
