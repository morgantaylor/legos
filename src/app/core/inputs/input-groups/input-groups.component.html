Input Group

An input-group component is explicitly used to combine button and input elements into a single element. They are by default inline like other input elements. Wrap both with a .input-group class for the desired effect. On mobile devices input-group components will become full width.

If you are trying to unite any other elements such as:

  • label + input
  • label + checkbox/radio

That should be done with the form-group class. See the form component page for more details on how to implement.

Demo:
Markup:
<div class="input-group">
  <input type="text" class="input input-group__input">
  <button type="submit" class="btn input-group__btn btn--blue">Submit</button>
</div>
