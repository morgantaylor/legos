Table

A table component is used for tabular data, not layout. The default <table> element has no column dividers, only row dividers.

Best Practices: Use the <table> element and not a class name. For accessibility if you are going to be displaying tabular data use <table>, <thead>, <tbody>, <tr>, <td>, and <caption> elements where appropriate. Screen readers and assistive technologies cannot read class names therefore don't know how to translate their functions.

Demo:
This is an example of a table caption
# Header Column 1 Header Column 2 Header Column 3
1 Body Column 1 Body Column 2 Body Column 3
2 Body Column 1 Body Column 2 Body Column 3
3 Body Column 1 Body Column 2 Body Column 3
Markup:
<table>
  <caption>table caption</caption>
  <thead>
    <tr>
      <td>#</td>
      <td>Header Column 1</td>
      <td>Header Column 2</td>
      <td>Header Column 3</td>
    </tr>
  </thead>
  <tbody>
    <tr>
      <td>1</td>
      <td>Body Column 1</td>
      <td>Body Column 2</td>
      <td>Body Column 3</td>
    </tr>
    ...
  </tbody>
</table>
