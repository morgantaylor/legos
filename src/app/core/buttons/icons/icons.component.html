Button Icon

Incorporate Font Awesome icons into buttons or use an icon as a button. If you have a button that has an icon and text wrap the text in a span element to allow for sufficient padding between the icon and the text.

When using an icon as the only content in a button add an aria-label to assist screen readers in understanding the purpose of the button.

Attributes:
  • aria-label : defines a label for the element (only used when label text not visible on screen)
Demo:
Markup:
<button type="button" class="btn btn--blue">
  <i class="icon fas fa-cog" aria-hidden="true"></i>
  <span>Settings</span>
</button>
<button type="button" class="btn btn--blue" aria-label="" title="">
  <i class="icon fas fa-cog" aria-hidden="true"></i>
</button>

Upload Button

The upload button is special. To create an upload button we combine a few different elements. The label element has a unique .btn-upload class on it that makes it look like a button. Note that the button skin class needs to go on it.

Demo:
Markup:
<div type="button" class="btn-upload-container">
  <input type="file" name="uploadFile" id="upload" class="inputfile" 
    tabindex="0" 
    accept=".xlsx" 
    required="required" 
    aria-required="true" 
    aria-invalid="true">
  <label for="upload" class="btn-upload btn--blue">
    <i class="icon fas fa-upload" aria-hidden="true"></i>
    <span>Choose a file</span>
  </label>
</div>
